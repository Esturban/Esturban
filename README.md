## How this profile works

This README is generated by a Go program running on a 6-hour cron via GitHub Actions. It reads `profile.json` for structured content, fetches RSS feeds from Hugging Face, OpenAI, DeepMind, Ollama, Latent.Space, and Google AI, scores each item by recency and relevance, and writes the AI Radar section with the top picks. Star counts are fetched live from the GitHub API at render time.
